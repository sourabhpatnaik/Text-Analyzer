# Text Analyzer

A Python-based text analysis tool that provides statistics about your text.

## Features
- Character count
- Word count
- Sentence count
- Digit count
- Average word length
- Average sentence length

## How to Run
```
python main.py
```

## Project Structure
- `main.py` - Entry point
- `menu.py` - Menu and display
- `text_analyzer.py` - Analysis functions
